### 1.1.3 选择 CityU(DG) 的核心原因

- 离家近（东莞），生活和实习衔接方便  
  有宿舍、学费和生活成本更低，去海外或香港成本更高

## 1.2 个人建议（更口语一点）

- 先想清楚你读研的目标，越早越不焦虑：  
   - 水个专业好找工作、不一定对口；
   - 找对口专业对口就业；
   - 准备读 PhD 但不想国内学制；
   - 或者就是想出国体验
- 申请定位：  
	- GPA 和雅思够的话可以冲高；
	- 不够就用项目/竞赛/实习补软背景，广撒网  
	- 选港/新/澳/英/美主要看预算和需求（生活、认可度、未来打算）
- 雅思/托福尽量早点刷出来：  
  不同项目要求不一样，早点刷出来省心；英语提升多靠长期接触（论文、博客、论坛）
- 预算够又想省事，可以找中介：  
  先帮你理清脉络和定位；DIY 需要时间沉淀  
  我前期找过指南者，需要的话可以找我来推荐负责人
- 小红书有很多经验贴，可以当参考


# 2. 未来实习就业（计算机相关）
